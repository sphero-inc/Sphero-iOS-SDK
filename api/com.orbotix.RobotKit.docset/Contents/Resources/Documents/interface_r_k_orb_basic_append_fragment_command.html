Detailed Description

Command to append complete lines of orb basic code to ball memory. Fragment data should be ascii encoded string. Sending an orbBasic program to Sphero involves appending complete lines to any existing ones in the specified storage area. Complete lines are required. A line begins with a decimal line number followed by a space and is terminated with a <CR>.
