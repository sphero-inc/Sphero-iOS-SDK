RKBatteryPowerState state

Detailed Description

To enable these notification, use the RKSetPowerNotificationCommand to initiate the notifications. The robot will send the notifications at intervals of 10 seconds with the battery state. The notification is received in client code by registering as an observer for data streaming events.
